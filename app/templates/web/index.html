Getting Started
Epic EHR Launch (Recommended)

Launch this application from within Epic's EHR system with bidirectional coding enabled to access the full HL7 messaging interface. This provides getMessage and setMessage endpoints for complete workflow automation.

Standalone Testing

The HL7 message parser is available for standalone testing and development without Epic connectivity. Use this mode to test message parsing logic and validate HL7 message structures.

Note: This application requires Epic EHR launch with bidirectional coding configuration to access the full HL7 messaging interface. Contact your Epic administrator to enable bidirectional coding for this application.
